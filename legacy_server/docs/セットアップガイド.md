# 転職先企業リサーチ用 NotionDB セットアップガイド

## 📦 ファイル構成

1. **転職リサーチDB_インポート用.csv** - データベースのプロパティ定義とサンプルデータ
2. **求人ページテンプレート.md** - 各企業ページの本文テンプレート

---

## 🚀 セットアップ手順

### Step 1: CSVファイルをNotionにインポート

1. Notionを開く
2. 左サイドバーの「+ ページを追加」をクリック
3. 「インポート」を選択
4. 「CSV」を選択
5. `転職リサーチDB_インポート用.csv` をアップロード
6. 「転職リサーチDB」などの名前をつける

### Step 4: ページテンプレートを設定

1. データベースの右上「新規」ボタンの横の「▼」をクリック
2. 「+ 新規テンプレート」を選択
3. テンプレート名を「求人詳細テンプレート」に設定
4. `求人ページテンプレート.md` の内容をコピペ
5. 「戻る」で保存

### Step 5: ビューを作成

データベース上部の「+ ビューを追加」から以下を作成：

**📋 全求人一覧（テーブルビュー）**
- すべての列を表示
- フィルターなし

**🎯 応募中（ボードビュー）**
- グループ化：応募ステータス
- カンバン形式で進捗管理

**⭐ 高評価のみ（テーブルビュー）**
- フィルター：総合評価 が ⭐⭐⭐⭐ または ⭐⭐⭐⭐⭐

**🚨 要注意企業（テーブルビュー）**
- フィルター：⚠️通勤1時間超 または ⚠️長時間労働 または ⚠️給与不公平感 または ⚠️雇用不安定 が チェック済み

**📅 カレンダー（カレンダービュー）**
- 日付プロパティ：次回アクション日

**💰 年収順（テーブルビュー）**
- 並び替え：想定年収（上限）降順

## 💡 活用のコツ

### 毎日やること
1. 新しい求人を見つけたらすぐ登録
2. 応募ステータスを更新
3. 次回アクション日を設定

### 週1回やること
1. 全体を見直して優先度を再評価
2. 応募漏れがないかチェック
3. 要注意企業ビューを確認

### 面接前にやること
1. 該当企業のページを開いて情報を復習
2. 想定質問への回答を準備
3. 逆質問リストをチェック

### 面接後にやること
1. 所感を必ず記録（記憶が新鮮なうちに）
2. 応募ステータスを更新
3. 次回アクション日を設定

---

## ⚠️ 注意事項

- サンプルデータ（3件）は削除してから使用してください
- CSVのエンコードはUTF-8です
- Notionの無料プランでも使用可能です
