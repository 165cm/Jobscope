## 4. 推奨アクションプラン

### Phase 1: リブランディング（即時）

#### 名称変更

現在の「Jobscope」は以下の問題がある：
- 機能が伝わらない（AI解析・Notion連携が不明）
- SEO競合が多い（JobScoreと混同）
- 差別化できていない

**推奨候補：**

| 順位 | 名称 | 理由 |
|------|------|------|
| 1位 | **JobPocket** | 覚えやすい、収集感、親しみやすい |
| 2位 | **JobSync** | Notion連携の「同期」感 |
| 3位 | **Jobry** | ユニーク、SEO独占可能 |

### Phase 2: 販売準備（Week 1-2）

1. **ランディングページ作成**
   - 日本語LP
   - 価値提案を明確に（「1クリックで求人情報をNotionに整理」）

2. **Notion DBテンプレート最適化**
   - Green, Wantedly, doda, BizReach対応
   - カンバン、カレンダー、ギャラリービュー設定
   - サンプルデータ入り

3. **販売チャネル設定**
   - Gumroad でストア開設
   - $5 スターターパック

### Phase 3: ローンチ（Week 3-4）

**価格戦略：**

| プラン | 価格 | 内容 |
|--------|------|------|
| Early Bird | $3 | 先着100名限定 |
| **Starter** | **$5** | 拡張 + Notion DB + ガイド |

**バンドル内容：**
- Chrome拡張機能
- 最適化されたNotion DBテンプレート
- クイックスタートガイド（日本語）
- 30日間メールサポート

### Phase 4: マーケティング（継続）

**チャネル：**
- Twitter/X（転職・キャリア系アカウント）
- Qiita/Zenn（技術記事）
- note（使い方・体験談）
- 転職系コミュニティ

**コンテンツ戦略：**
- 「Notionで転職活動を効率化する方法」
- 「100社応募して分かった求人管理のコツ」
- ユーザー事例・インタビュー

### Phase 5: 成長と拡張（6ヶ月後〜）

```
成功条件達成後（収益$3,000+/年）：

Option A: 縦展開
  → 面接対策AI、年収交渉サポート、企業分析機能

Option B: 横展開
  → 不動産情報収集、投資情報整理など
  → 各ニッチ向けに別ブランドで展開

※ 汎用化は「ニッチで成功してから」検討
```

## 5. 顧客満足度最大化施策

### $5バンドルに含めるべきもの

| 項目 | 理由 |
|------|------|
| Notion DBテンプレート（最適化済み） | すぐに使える状態 |
| クイックスタートガイド（日本語） | APIキー設定の壁を下げる |
| 動画チュートリアル（5分） | 視覚的に理解 |
| サンプルデータ | 使用イメージが湧く |
| 30日間メールサポート | 初期つまずきのフォロー |

### オンボーディングフロー

```
Day 0:  購入完了メール + アクセス情報

Day 3:  「初めての解析できましたか？」チェックイン
Day 7:  活用Tips + 応用テクニック
Day 14: フィードバック＆レビュー依頼
Day 30: エンゲージメントメール
```
