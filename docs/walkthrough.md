# 🚀 Jobscope Chrome拡張機能 - 導入ガイド

Jobscope Chrome拡張機能のインストール、設定、使い方について説明します。

## 📦 インストール手順

1.  **プロジェクトのビルド** (完了済み)
    ビルド成果物は `/dist` ディレクトリに生成されています。

2.  **Chromeへの読み込み**
    1.  Chromeを開き、アドレスバーに `chrome://extensions` と入力して移動します。
    2.  右上の **デベロッパーモード** をONにします。
    3.  **パッケージ化されていない拡張機能を読み込む** (Load unpacked) をクリックします。
    4.  プロジェクトフォルダ内の **`dist`** フォルダを選択します (`Jobscope/dist`)。

## ⚙️ 初期設定

1.  ブラウザのツールバーにある **Jobscopeアイコン** をクリックします（ピン留めしておくと便利です）。
2.  画面右上の **設定(歯車)アイコン** をクリックします。
    *   (または、初回起動時に表示される「Open Settings」ボタンをクリックします)
3.  設定画面が開きます。
4.  以下のAPIキーを入力します：
    *   **OpenAI API Key**: `sk-` で始まるキー
    *   **Notion Integration Token**: `secret_` で始まるトークン
    *   **Notion Database ID**: データベースURLに含まれる32桁のID
5.  **Save Settings** をクリックして保存します。

## 🕵️ 使い方

### 1. ゼロクリック解析 (Zero-Click Analysis)
求人サイト（Green, Wantedlyなど）の求人ページを開きます。
**Jobscopeアイコン** をクリックしてサイドパネルを開きます。
*   **Analyze This Page** ボタンをクリックします。
*   AIがページを読み込み、企業名・年収・スキルなどを自動抽出します。

### 2. 編集と確認 (Edit & Review)
解析結果がサイドパネルに表示されます。
*   **項目の編集**: 企業名、年収、雇用形態などのフィールドをクリックして直接編集できます。
*   **マッチ度**: AIの判定を手動で変更することも可能です。

### 3. Notionへ保存 (Save to Notion)
*   **Save to Notion** ボタンをクリックします。
*   保存が成功するとメッセージが表示され、「Open in Notion」リンクからNotionページを開けます。
*   保存後はボタンが **Update Notion** に変わります。

### 4. データの更新 (Update Existing)
*   サイドパネルで内容を修正した場合、**Update Notion** をクリックすれば、同じNotionページに変更内容が上書き同期されます。

### 5. 🆕 URL重複チェック (Duplicate Detection)
*   既にNotionに登録済みの求人ページを分析すると、**青いアラートバー**で通知されます。
*   自動的に「更新モード」に切り替わり、既存のエントリを上書き更新できます。
*   ×ボタンをクリックすると、新規として保存することも可能です。

## ✨ 高度な機能 (Advanced Features)

### ⚡ AIモデルの選択
設定画面 (API Tab) で、解析に使用するAIモデルを選択できます。
*   **GPT-4o mini**: デフォルト。非常に安価で高速です。
*   **GPT-4o**: 複雑な求人票の解析に適した高精度モデルです。
*   **カスタムモデル**: リスト横の **「＋」ボタン** から、将来リリースされる新しいモデル（例: gpt-5-nano）などを追加できます。Input/Outputコストを入力すると、自動でコスト試算が行われます。

### 📝 生成ルールのカスタマイズ
設定画面の **「✨ プロンプト編集」** タブから、AIへの指示を細かく調整できます。
*   **コンテンツ生成ルール**: Notionのページ本文（Markdown）の構成を変更できます（見出し、箇条書きのスタイルなど）。
*   **抽出ルール**: 各プロパティ（会社名、スキルなど）の抽出基準を調整できます。

## 🛠 トラブルシューティング

*   **"No response from content script"**: 求人ページをリロード（再読み込み）してみてください。拡張機能をインストールした後、既に開いていたページにはスクリプトが読み込まれていない場合があります。
*   **APIエラー**: 設定画面 (Options) でAPIキーが正しく入力されているか確認してください。
