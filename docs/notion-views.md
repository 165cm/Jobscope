# おすすめ Notion ビュー設定ガイド

求人情報を効率的に比較・検討するための、おすすめのビュー設定（表示形式）をまとめました。
Notionデータベースの「＋ビューを追加」から作成してください。

## 1. 🏆 総合ランキング (テーブルビュー)
まずはこれ！評価が高く、条件が良い順に並べる基本のビューです。

*   **表示形式**: テーブル
*   **ソート順 (並び替え)**:
    1.  `Rating` (総合評価) : **降順** (星が多い順)
    2.  `Match` (マッチ度) : **昇順** (◎が上に来るように調整 ※Notionの仕様による)
    3.  `Salary (Max)` (年収上限) : **降順**
*   **フィルター**:
    *   `Match` : `×` と `空欄` を除外 (Is not empty, Is not ×)

## 2. 💰 年収 × マッチ度 (ボードビュー)
「スキルは合うけど給料が...」「給料はいいけどスキルが...」のバランスを一目で確認できます。

*   **表示形式**: ボード
*   **グループ化**: `Match` (スキルマッチ度)
    *   表示順: ◎完全一致 → ○ほぼ一致 → △一部一致
*   **プロパティ表示設定**:
    *   `Salary (Min)` - `Salary (Max)` (数値が見えるようにON)
    *   `Company Name`
    *   `Rating`

## 3. 🏠 働き方・環境 (ボードビュー)
リモートワークの可否や勤務地で分け、働きやすさを重視して探す時に便利です。

*   **表示形式**: ボード
*   **グループ化**: `Remote` (リモートワーク)
    *   表示順: フルリモート → 週3-4日 → 週1-2日 → なし
*   **プロパティ表示設定**:
    *   `Location` (勤務地)
    *   `Commute Time` (通勤時間)
    *   `Autonomy` (裁量権あり - チェックボックス)

## 4. 🏢 企業安定性チェック (テーブルビュー)
大手企業や歴史のある企業を探したい時に使います。

*   **表示形式**: テーブル
*   **ソート順**:
    1.  `Employees` (従業員数) : **降順**
    2.  `Avg Age` (平均年齢) : **降順**
*   **プロパティ表示設定**:
    *   `Employees`, `Avg Age` を左側に配置して見やすくする

## 5. 🚩 要注意リスト (リストビュー)
「ブラック」「長時間労働」などのフラグが立った求人だけを抽出し、本当にエントリーして良いか再考するためのリストです。

*   **表示形式**: リスト (またはギャラリー)
*   **フィルター**: (以下のいずれか条件なら表示)
    *   `Overwork` (長時間労働) : チェックあり **OR**
    *   `Long Commute` (通勤遠い) : チェックあり
*   **色分け**:
    *   アラート用に、背景色を変えるなどはできませんが、タイトルを見るだけで注意喚起になります。
